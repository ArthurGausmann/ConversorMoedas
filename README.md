# Every Penny

Every Penny is a simple and efficient currency converter web application built with React and Vite. It allows users to convert between different currencies (BRL, USD, and EUR) in real-time using the Frankfurter API.

![Every Penny Logo](/public/logo.png)

## Features

- **Real-time Conversion**: Fetch latest exchange rates from the Frankfurter API.
- **Support for Multiple Currencies**: Convert between BRL (Brazilian Real), USD (US Dollar), and EUR (Euro).
- **Responsive Design**: Clean and simple user interface.
- **Input Validation**: Prevents converting a currency to itself and handles API errors.

## Tech Stack

- **Framework**: [React](https://react.dev/) (v19)
- **Build Tool**: [Vite](https://vite.dev/)
- **API**: [Frankfurter API](https://www.frankfurter.app/)
- **Styling**: CSS3
- **Linting**: ESLint

## Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Usage

1. Select the base currency from the first dropdown.
2. Enter the amount you wish to convert.
3. Select the target currency from the second dropdown.
4. Click the "Converter" button to see the result.
